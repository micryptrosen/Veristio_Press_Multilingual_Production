## Reviewer Role Requirements

A native-language reviewer for this pilot must be able to review Spanish for naturalness, clarity, tone, register, cultural fit, and faithful preservation of meaning while respecting the locked English source identity.

The reviewer must understand that this is a language-quality gate only. It is not technical approval, final sync/proof approval, metadata approval, ISBN approval, publication approval, or release approval.

## Repair Workflow

1. Operator identifies a candidate reviewer outside this repository process.
2. Candidate is screened against `docs/reviewer_program/SPANISH_NATIVE_REVIEWER_QUALIFICATION_CRITERIA_V1.md`.
3. Candidate receives `docs/reviewer_program/SPANISH_NATIVE_REVIEWER_ONBOARDING_PACKET_V1.md`.
4. Candidate completes or accepts the evaluation sample protocol in `docs/reviewer_program/SPANISH_NATIVE_REVIEWER_EVALUATION_SAMPLE_PROTOCOL_V1.md`.
5. Operator records assignment evidence using `docs/reviewer_program/SPANISH_NATIVE_REVIEWER_ASSIGNMENT_RECORD_TEMPLATE_V1.md`.
6. Only after the assignment record is completed may the Book 1 Chapter 1 native review execution corridor reopen.

## Evidence Required Before Review Execution

- Reviewer legal or working name.
- Reviewer role: native-language reviewer.
- Native Spanish fluency basis or equivalent professional review basis.
- Locale or neutral-Spanish expectation acknowledgement.
- Confidentiality/IP acknowledgement.
- AI-use boundary acknowledgement.
- Evaluation sample outcome or operator waiver reason.
- Assignment date.
- Scope: Book 1 Chapter 1 Spanish Draft V2 native-language review.
- Signoff that publication/release remains blocked.

## Escalation Path

If a qualified reviewer cannot be found, keep the lane at `HOLD_FOR_OPERATOR_REVIEWER_SELECTION` and do not execute native review. If reviewer qualification is uncertain, escalate to operator decision before assigning the reviewer. If the reviewer requests source changes, route the request through Book 1 authority and translation change sync rather than editing Book 1 from this repo.

## Boundary

This plan does not assign a reviewer, does not execute native-language review, does not create native-review findings, and does not approve the Spanish draft. Publication remains blocked until documented native review occurs and all later technical, final sync/proof, metadata, ISBN, proof, publication, and release gates pass.

## Authority Boundary

Book 1 remains the canonical English owner for manuscript text, readiness, claims, citations, and permissions. Multilingual Production owns Spanish reviewer program readiness, reviewer assignment evidence templates, packet routing, tracker posture, and downstream language-edition workflow gates.
